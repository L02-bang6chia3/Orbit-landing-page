# Orbit Landing Page

A modern React landing page built with Vite, Tailwind CSS, and Framer Motion animations. This project showcases a responsive and interactive landing page with smooth animations and a clean design, perfect for showcasing products or services.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```
